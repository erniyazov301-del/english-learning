# Lumina English Learning App

A minimalist, high-performance web application for mastering English, designed with a clean **Notion-inspired** aesthetic.

## 🚀 Features
- **First Launch Setup:** Personalized English level (A1-C2) and typography settings.
- **Grammar Translation Tests:** 500 sentences per level with instant feedback and progress tracking.
- **Vocabulary Flashcards:** Level-based word queues with "Repeat" or "Know" logic.
- **Grammar Reference:** Concise lessons on core English structures (Tenses, Passive Voice, Conditionals).
- **Persistent Progress:** All data and settings are saved locally in the browser via `localStorage`.

## 🛠️ Technical Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript.
- **Deployment:** Hosted via GitHub Pages.
- **No Dependencies:** Runs directly in the browser without installation or external frameworks.

## 📖 How to Use
1. Open the [Live Demo](YOUR_LINK_HERE).
2. Complete the initial setup by choosing your level.
3. Use the **Tests** tab to practice translation or the **Cards** tab for vocabulary.
4. Your progress is saved automatically—you can close the tab and return later.
